# Mini Player

Generate a Minecraft datapack that shrinks players to 1/5th size with balanced attributes.

## Quick Start

1. Clone and Run the generator:
```bash
python mini_player.py
```

2. Place `mini_player.zip` in your world's `datapacks` folder

3. Use in-game commands:
   - `/function mini:shrink` - Shrink player
   - `/function mini:reset` - Reset to normal
